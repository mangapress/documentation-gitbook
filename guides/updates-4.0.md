# New in Manga+Press 4.0

* New actions and filters have been added.

* Child-themes removed completely. In their place, a Template API has been created to allow theme authors to fully
support Manga+Press

* New content-type has been added, called Comic Pages. Comic Pages will be used for Latest Comic and Comic Archives pages in
new installations. Backwards compatibility for default WordPress pages will be maintained.

* Complete refactor of Manga+Press's code-base
